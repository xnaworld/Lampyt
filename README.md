# Lampyt 

### Overview ###
 
Lampyt is a tool that can convert a file into a pay file. Through a pay file, anyone who wants the data inside it, should pay coins in blockchain to obtain the data.

### Environment ###
requirement:
   .net framework 4.7.2
   
   NET 6.0 Desktop Runtime (v6.0.22)
   
   Win10(recommend)
